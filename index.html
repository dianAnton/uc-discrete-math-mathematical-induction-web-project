LENGUAJE MATEMÁTICO

Notación necesaria

  • x ∈ B (x pertenece a B)
  • x ∈ B (x no pertenece a B)
  • ∃x (Existe x)
  • ∀x (Para todo x)
  • A ⊆ B (A es subconjunto de B)
  • A ⊊ B (A es subconjunto propio de B)

DEFINICIÓN INFORMAL DE LOS NÚMEROS NATURALES

Los números naturales, denotados por ℕ, son los números que sirven para contar los elementos de un conjunto.

AXIOMAS DE PEANO

  1. El 0 ∈ ℕ.
  2. Si n e ℕ, entonces (n + 1) ∈ ℕ donde n + 1 es el sucesor de n.
  3. Todo n ∈ ℕ, tal que n ≠ 0 tiene un antecesor en ℕ.
  4. Principio del buen orden: Todo subconjunto no vacio A ⊆ ℕ tiene un menor elemento.
